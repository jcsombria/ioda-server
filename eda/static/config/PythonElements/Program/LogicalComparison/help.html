LogicalComparison

Does one of several possible logical comparisons

Operations supported:
  • >: checks if operand 1 is strictly bigger than operand 2
  • >=: checks if operand 1 is bigger than or equal to operand 2
  • <: checks if operand 1 is strictly smaller than operand 2
  • <=: checks if operand 1 is smaller than or equal tooperand 2
  • ==: checks if operand 1 is equal to operand 2
  • !=: checks if operand 1 is different from operand 2
